## Key Decisions Embedded
1. **7-stage sequential pipeline** (not event-driven Phase 1)
2. **All stages idempotent** — re-runs skip completed work
3. **Services wrap Azure SDKs** — pipeline never calls Azure directly
4. **Managed Identity everywhere** — zero secrets in code
5. **JSON mode LLM output** — structured extraction in one call
6. **Seed glossary + LLM detection** — ~60 known terms + dynamic capture
7. **ePub-first, PDF from HTML** — single canonical source
8. **PostgreSQL (persistent) + Redis (ephemeral)** — clear boundaries
9. **Python 3.12+, src layout, hatch, ruff, pytest** — modern stack
10. **PEP 563 forward compatibility** — `from __future__ import annotations` everywhere
